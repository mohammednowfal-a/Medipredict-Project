ℹ️ About MediPredict

Symptom-Based Disease Prediction System

🎯 Purpose

MediPredict is an educational project demonstrating AI-assisted disease prediction using symptom matching. Built as a final-year project using Python Flask + SQLite.

🛠️ Tech Stack

  • 🐍 Python 3 + Flask
  • 🗄️ SQLite (user accounts, history)
  • 🌐 HTML5 + CSS3 + Vanilla JS
  • 📊 CSV Dataset (25+ diseases)

✨ Features

  • 🩺 77+ Symptom checker with categories
  • 💊 Medicine suggestions
  • 🏥 Nearby hospital finder
  • 📅 Appointment reminder
  • 🧬 BMI calculator
  • 📜 Diagnosis history
  • ⭐ Disease bookmarks
  • 🌐 Hindi/English toggle
  • 🌗 Dark/Light mode

⚠️ Disclaimer

This system is NOT a medical device. Predictions are based on a simple dataset. Always consult a qualified healthcare provider for diagnosis and treatment.
