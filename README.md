# MediPredict – AI Disease Prediction Web Application

## Overview

MediPredict is an AI-powered web application that helps users predict possible diseases based on symptoms. It also provides BMI calculation, hospital finder, user authentication, and medical history management.

## Features

- Disease Prediction
- User Login & Registration
- BMI Calculator
- Hospital Finder
- Medical History
- Bookmark Diseases
- Responsive User Interface

## Technologies Used

- Python
- Flask
- SQLite
- HTML5
- CSS3
- JavaScript
- REST APIs

## Folder Structure

```
medipredict_v2/
├── app.py
├── requirements.txt
├── data/
├── static/
└── templates/
```

## Installation

1. Clone the repository

```
git clone https://github.com/mohammednowfal-a/Medipredict-Project.git
```

2. Install dependencies

```
pip install -r requirements.txt
```

3. Run the application

```
python app.py
```
